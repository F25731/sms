# 短信接码网站

轻量 Node 版本，前端是 iOS 玻璃拟态卡密兑换页，后端直接代理 sms.oapi.vip 的前端接口，客户可以直接使用从原网站购买的卡密。

## 功能特点

- ✅ 完整复刻原网站功能（CDK验证、获取验证码、换号）
- ✅ 优化的前端体验（防抖、状态管理、错误恢复）
- ✅ 自动轮询验证码（3秒间隔）
- ✅ 页面可见性智能管理（切换标签时暂停轮询）
- ✅ 超时保护和重试机制
- ✅ 会话自动清理

## 本地启动

```bash
npm start
```

访问 `http://localhost:3000`。

## Docker部署

```bash
docker build -t sms-glass-relay .
docker run -d --name sms-glass-relay -p 3000:3000 sms-glass-relay
```

## 环境变量

- `PORT` - 服务端口（默认3000）
- `SESSION_TTL_MINUTES` - 会话有效期（默认30分钟）
- `MIN_POLL_INTERVAL_SECONDS` - 最小轮询间隔（默认3秒）

## 页面流程

1. 用户输入卡密并点击兑换。
2. 服务端调用 `check_cdk` 获取手机号和状态。
3. 兑换成功后跳转到 `/detail.html?session=...`。
4. 详情页每 3 秒调用 `/api/sms` 查询验证码。
5. 未收到时保持等待，收到后停止轮询并显示短信全文与验证码。
6. 支持手动刷新和换号功能。
